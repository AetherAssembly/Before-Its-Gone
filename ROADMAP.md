## Current: v0.5.0

### Smart features & data management (all platforms)

- ✅ Native desktop app (Electron + React) for Linux, macOS, and Windows
- ✅ Barcode scanning and profile storage
- ✅ Shelf-life and expiry date tracking
- ✅ Desktop notifications for expiring items
- ✅ Color-coded expiry cards
- ✅ Search by name, barcode, or category
- ✅ Filter by storage location
- ✅ Sort by expiry date, date added, or name
- ✅ Export to JSON and CSV
- ✅ Import from JSON export
- ✅ Clear all data with confirmation
- ✅ Stats dashboard (total, expiring this week, expired)
- ✅ Categories for items
- ✅ Item history and quick-add buttons
- ✅ Quantity depletion tracking with low-stock notifications

---

## v0.6.0 — Editing & Organization

### Focus: In-place editing and richer tagging

- Edit existing inventory items without delete-and-re-add
- Multiple tags per item (comma-separated)
- Bulk actions (select multiple → delete, move location)
- Estimated: Q4 2026

---

## v1.0.0 — Stability & Polish

### Focus: Feature-complete, stable release

- Performance optimization for large inventories
- UI refinements and accessibility improvements
- Comprehensive documentation
- Community feedback integration
- Estimated: Q2 2027
